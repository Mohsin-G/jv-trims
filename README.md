# JV Trims

Professional barbershop website for JV Trims, Manchester.

## Live Site
[jv-trims.co.uk](https://jv-trims.co.uk)

## Tech Stack
- [Next.js](https://nextjs.org/) - React framework
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [Vercel](https://vercel.com/) - Hosting & deployment
- [Booksy](https://booksy.com/) - Booking integration

## Features
- Mobile-first responsive design
- Services & pricing
- Photo gallery
- Booksy booking integration
- SEO optimised for local search

## Getting Started
Clone the repo and install dependencies:

```bash
git clone https://github.com/Mohsin-G/jv-trims.git
cd jv-trims
npm install
npm run dev
```

## Deployment
Deployed automatically via Vercel on every push to `main`.
